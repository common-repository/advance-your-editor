Advance the classic editor and the block editor (Gutenberg) with user friendly additional functionalities.



== Description ==

Advance your Editor add additional functionalities to Classic Editor and Block Editor (Gutenberg). It can be used to insert tables, font settings, search and replace and  much more.



== Installation ==

* Upload the Advance your Editor plugin to the `/wp-content/plugins/` directory

* Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= How to use Advance your Editor? =
* Once plugin is installed, you can go to edit your posts or page. It will be available. 

= Can I edit settings? =
* Yes, you can edit settings for plugin in Settings > Advance Your Editor menu.

= Can I use it to create tables? =
* Yes

= Can I set Font Family and Font Size? =
* Yes

= Which Editors are supported? =
* It supports classic editor and block editors (Gutenberg). 

= Do it have Search and Replace functionality? =
* Yes

= Can I insert lists in it? =
* Yes
